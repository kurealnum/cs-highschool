/:

Returns what is basically the home page, I.E. the index, help page, ect. Displays basic info on the API. There are URIs listed everywhere, just for a helping hand!

/ (Just the index)

/api/all_countries:

Returns all the countries supported/shown in this dataset. URI should look like this:

/api/all_countries

/api/avg_of_pollution:

Returns the average amount of pollution from all countries, for a certain kind of pollution, in a certain year. URI should look like this:

/api/avg_of_pollution/carbon_dioxide_co2_emissions_without_land_use_land_use_change_and_forestry_lulucf_in_kilotonne_co2_equivalent/2010

/api/years_supported:

Returns the years supported in the dataset. URI should look like this:

/api/years_supported

/api/types_of_emissions:

Returns the types of emissions that this dataset contains. URI should look like this:

/api/types_of_emissions

/api/pollution_info:

Returns the amount of pollution for each country, with the option to sort by "amount" of pollution in either ascending or descending. URI should look something like:

/api/pollution_info/asc/carbon_dioxide_co2_emissions_without_land_use_land_use_change_and_forestry_lulucf_in_kilotonne_co2_equivalent/2010

/api/h_l_polluting_countries:

Returns the highest or lowest polluting country for a certain year. URI should look like this:

/api/h_l_polluting_countries/lowest/carbon_dioxide_co2_emissions_without_land_use_land_use_change_and_forestry_lulucf_in_kilotonne_co2_equivalent/2000

/api/difference_in_pollution:

Returns the difference in pollution across a period of time for a certain country. Make sure years are in ascending order (i.e. 2000-2010, not 2010-2000). Also, note that the return value is the difference between the newest value and the oldest value. If it returns 0, you probably typed something in wrong. URI should look like this:

/api/difference_in_pollution/Australiacarbon_dioxide_co2_emissions_without_land_use_land_use_change_and_forestry_lulucf_in_kiotonne_co2_equivalent/2010-2012
